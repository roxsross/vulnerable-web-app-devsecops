🔓 Vulnerable Web App

⚠️ Esta aplicación contiene vulnerabilidades INTENCIONALES para fines educativos

🔐 Login (Broken Authentication)

💡 Hint: Las credenciales están hardcodeadas en el backend

🔍 Búsqueda de Usuarios (SQL Injection)

💡 Hint: El query se concatena directamente sin sanitización

💬 Comentarios (XSS)

💡 Hint: Intenta inyectar un script o img tag con onerror

👤 Ver Perfil de Usuario (IDOR)

💡 Hint: Cambia el ID para acceder a perfiles de otros usuarios sin validación

🖥️ Ping Server (Command Injection)

💡 Hint: Intenta inyectar comandos con ; o && o |

📚 Vulnerabilidades Incluidas

  • SQL Injection: Concatenación directa de input en queries
  • XSS: Renderizado de input sin sanitización (innerHTML)
  • Broken Authentication: Credenciales hardcodeadas, JWT sin expiración
  • Sensitive Data Exposure: API keys en el código fuente y responses
  • IDOR: Acceso a recursos sin validación de autorización
  • Security Misconfiguration: CORS abierto, headers de seguridad faltantes
  • Vulnerable Dependencies: lodash@4.17.15, axios@0.21.1, express@4.17.1
  • Command Injection: Ejecución de comandos sin sanitización

Ver VULNERABILITIES.md para detalles completos de cada vulnerabilidad.
